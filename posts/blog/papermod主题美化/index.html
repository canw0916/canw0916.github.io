Mac風格的code blocks#

在\themes\PaperMod\assets\css\extended底下新建一個.css檔，命名為mac-code-blocks.css

pre {
    position: relative;
    padding: 30px 10px 10px 10px;
}

.post-content pre code {
	font-family: Consolas;
}

pre::after {
    display: block;
    content: " ";
    position: absolute;
    border-radius: 50%;
    background: #ff5f56;
    width: 12px;
    height: 12px;
    top: 0;
    left: 12px;
    margin-top: 12px;
    -webkit-box-shadow: 20px 0 #ffbd2e, 40px 0 #27c93f;
    box-shadow: 20px 0 #ffbd2e, 40px 0 #27c93f;
}

添加utterances评论#

  1. 在github创建一个公共仓库用于存储评论内容

  2. 授权安装utterances

    • 进入utterances app 对自己所要保存评论的仓库授权
    • 授权之后再进去则可以配置参数

  3. 编辑comments配置,建议在根目录layouts\partials下创建comments.html文件并编辑：

    <!-- utterances comments-->
<div>
    <div class="pagination__title">
        <span class="pagination__title-h" style="font-size: 20px;">💬评论</span>
        <hr>
    </div>
    <div id="tcomment"></div>
    <script src="https://utteranc.es/client.js"
        repo="canw0916/utterances_comments"
        issue-term="title"
        theme="photon-dark"
        crossorigin="anonymous"
        async>
    </script>
</div>
    • repo换成自己的仓库名称,格式:用户名/仓库名
    • theme换成自己选择的主题名字

  4. 编辑config.yml,在params参数下添加(repo和theme换成自己的):

    utteranc:
    enable: true
    repo: "canw0916/utterances_comments" 
    issueTerm: "title"
    theme: "photon-dark"
