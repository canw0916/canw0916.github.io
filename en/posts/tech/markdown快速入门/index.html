Markdown快速入门

Markdown 是一种轻量级的标记语言，可用于在纯文本文档中添加格式化元素。Markdown 由 John Gruber 于 2004 年创建，如今已成为世界上最受欢迎的标记语言之一。

  • markdown专注于文字内容；
  • 纯文本，易读易写，可以方便地纳入版本控制；
  • 语法简单，没有什么学习成本，能轻松在码字的同时做出美观大方的排版。

常用语法速查表#

元素Markdown 语法说明
标题（Heading）# 标题文字几个#号就是几级标题 #和标题中间要有空格
粗体（Bold）**要加粗的文字**前后两个*号
斜体（Italic）*要加斜体的文字*前后一个*号
引用块（Blockquote）> blockquote一个>号 >和引用内容要有空格
有序列表（Ordered List）1. First item1. 内容 记得空格
无序列表（Unordered List）- First item- 内容 记得空格
代码（Code）code前后一个`
分隔线（Horizontal Rule）---
链接（Link）[title](https://www.example.com)
表格（Table）`Syntax
代码块（Fenced Code Block）```编程语言前后三个```
删除线（Strikethrough）~~The world is flat.~~前后两个~~
